InfoCallback 멤버 변수를 통해서 CS 연동 데이터를 Xml 형식으로 얻을 수 있습니다.

App에선 시나리오에 따라서 InfoCallback 함수를 호출해 주면 됩니다.

Parameters
[out]pBufXml을 얻을 버퍼 입니다.
[in]nMaxLenXml을 얻을 버퍼의 크기 입니다.
Returns
성공시: CS_HOOK_RETCODE 반환 실패시: nXmlBufferSize가 4k를 넘거나 0 이하인 경우 CS_HOOK_RETCODE_INVALID_ARG 반환합니다. Infocallback 멤버 변수가 nullptr인 경우 CS_HOOK_RETCODE_ERROR 반환합니다.
